# Engineering for AI - Spark Coursework

## Overview

This project demonstrates distributed data processing and analytical workflows using Apache Spark and PySpark.

The coursework focuses on scalable data engineering techniques, large-scale data processing, and analytical operations commonly used in AI and machine learning pipelines.

- Distributed data processing using Spark
- Data cleaning and preprocessing
- Spark DataFrame operations
- Data transformation and aggregation
- Analytical workflows for large datasets
- Scalable computation using PySpark
